📈 Personal Loan Customer Analysis & Prediction Dashboard

📊 Project Overview:-

This project presents an interactive dashboard created using Microsoft Excel to analyze customer behavior related to personal loan acceptance. The dashboard provides insights into how factors such as income, education level, credit card usage, and family size influence loan approval decisions.
The goal of this project is to transform raw banking data into meaningful visual insights that can help financial institutions understand customer profiles and improve loan approval strategies.

⚙️ Tools Used:-
Microsoft Excel
Pivot Tables
Data Visualization
Dashboard Design

Key Metrics:-
Total Customers: 5000
Customers Loan Accepted: 480
Loan Acceptance Rate: 9.6%
Loan Rejected: 4520
Average Credit Card Spending: ₹4.3K
Average Income: ₹108K

Dashboard Insights
The dashboard provides the following insights:
Age group distribution of customers who accepted personal loans
Loan acceptance based on average credit card usage
Relationship between education level and loan acceptance
Impact of income on loan approval
Loan acceptance based on family size
Comparison of mortgage, credit card, and securities account holders with loan acceptance

Filters Available
The dashboard includes interactive filters for:
Education Level
ZIP Code
These filters allow dynamic exploration of customer segments and loan acceptance patterns.

💡 Business Value
This analysis helps banks and financial institutions:
Identify customer segments more likely to accept personal loans
Understand financial characteristics of potential borrowers
Improve targeted marketing strategies for loan products

📌Conclusion
This Excel dashboard demonstrates how financial datasets can be analyzed visually to uncover patterns in customer loan behavior and support data-driven decision making.
